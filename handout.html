Getting Started (3 Ways)

1. Use Existing MCPs

  • Browse modelcontextprotocol.io
  • Install via Claude Desktop or your AI tool
  • Start using immediately

2. Identify Repetitive Tasks

  • Look for tasks you do multiple times a week
  • Focus on data aggregation and formatting tasks
  • Consider multi-source documentation workflows

3. Build Your Own

  • Use MCP SDK (TypeScript, Python)
  • Follow templates and guides
  • Publish to registry
