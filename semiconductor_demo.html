Comprehensive PYNQ-Z1 Setup Guide

A complete, step-by-step wizard to get your PYNQ board up and running smoothly.

Setup Progress

Welcome!

This guide provides a comprehensive walkthrough for setting up your PYNQ-Z1 board. PYNQ is an open-source project from Xilinx that makes it easy to design with Xilinx Zynq SoCs.

Follow the steps in the sidebar to prepare your SD card, set up the hardware, and connect to your board.

1. What You Need

Before you begin, make sure you have all the required hardware. Check off each item as you confirm you have it.

2. Get the PYNQ-Z1 Image

You need to download the official PYNQ image for your Z1 board. This file contains the complete operating system and all the necessary PYNQ software.

Download PYNQ-Z1 Image

After Downloading:

The downloaded file will be a compressed archive (e.g., a .zip or .xz file). You must extract it to get the .img file, which you will write to your SD card in the next step.

3. Write the SD Card Image

Now you need to write the .img file to your Micro SD card. This process will erase everything on the card.

We recommend using a tool like Balena Etcher, as it's simple and works on Windows, macOS, and Linux.

  1. Insert the Micro SD card into your computer.
  2. Open Balena Etcher (or your chosen imaging tool).
  3. Select Image: Choose the PYNQ .img file you extracted.
  4. Select Drive: Carefully choose your Micro SD card. Be very careful to select the correct drive!
  5. Flash! Click the button to write the image to the card. This may take several minutes.

4. Setup the Board Hardware

With your SD card ready, it's time to configure the physical board jumpers.

1

Safely eject the Micro SD card from your computer and insert it into the Micro SD card slot on the PYNQ-Z1 board.

2

Set the Boot Jumper (JP4) to the SD position. This tells the board to boot from the SD card.

3

Set the Power Jumper (JP5) to the USB position. This allows the board to be powered from the Micro-USB cable.

5. Connect and Power On

You're almost there! Now, let's connect everything and power up the board.

  1. Connect the Micro-USB cable from your PC to the USB/UART port on the board.
  2. Connect the Ethernet cable. The other end can go to your PC, a router, or a corporate network.
  3. Connect the power adapter to the board to turn it on.

Boot Sequence

After powering on, you will see some LEDs light up. The board is fully booted when the two blue LEDs and the four yellow/green user LEDs are on. This can take up to 3 minutes.

6. Access Jupyter Lab

Once the board has booted, you can access the PYNQ environment through your web browser. The method depends on how you connected the Ethernet cable.

Option A: Connected to a Network Router

Open a browser and navigate to http://pynq:9090. If your computer supports mDNS, this should work automatically.

Option B: Connected Directly to a PC

You must manually configure your PC's network settings. Set the IP address to 192.168.2.1 and the subnet mask to 255.255.255.0. Then, navigate to http://192.168.2.99:9090.

Option C: Connected to a Corporate Network

The board will be assigned an IP by DHCP. You will need to find this IP address. You can do this by connecting to the board via a serial terminal (see Troubleshooting) and running ifconfig.

Login Information

When prompted for a password, use the default:

xilinx

7. Troubleshooting

If you encounter issues, here are some common solutions.

Connecting to a Serial Terminal

If you can't connect via the network, you can use a serial terminal (like PuTTY or Tera Term) to get command-line access. Find the COM port for the USB/UART device in your computer's device manager and connect with the following settings:

  • Baud rate: 115200
  • Data bits: 8
  • Stop bits: 1
  • Parity: None
  • Flow control: None

You can log in with username xilinx and password xilinx.

Finding the IP Address

Once connected via serial, run ifconfig eth0 to see the IP address assigned to the board's Ethernet port.

Setup Complete!

Congratulations! Your PYNQ-Z1 board is now set up and ready to use.

You can now start exploring the example notebooks in Jupyter Lab to learn what PYNQ can do.
